# runs

Scripts de configuración para mi entorno de trabajo.

## Uso

### Windows
```powershell
Set-ExecutionPolicy Unrestricted
.\windows.ps1
```

### Linux (WSL Ubuntu)
```bash
bash wsl-ubuntu.sh
```

### Fedora
```bash
bash fedora.sh
```

### Linux Mint
```bash
bash linux-mint.sh
```

### Arch Linux
```bash
bash arch-linux.sh
```

## Archivos

- `windows.ps1` - Windows
- `wsl-ubuntu.sh` - WSL Ubuntu
- `fedora.sh` - Fedora
- `linux-mint.sh` - Linux Mint
- `arch-linux.sh` - Arch Linux
