# runs

Scripts de configuración para mi entorno de trabajo.

## Uso

### Windows
```powershell
Set-ExecutionPolicy Unrestricted
.\windows.ps1
.\scoop.ps1
```

### Linux (WSL Ubuntu)
```bash
./wsl-ubuntu.sh
```

### Fedora
```bash
./fedora.sh
```

### Linux Mint
```bash
./linux-mint.sh
```

## Requisitos

- **Windows**: Winget instalado
- **Linux**: sudo/root
